Notes on the Sports Store App

Chapter 12 - Security

In Ch. 12 of Pro ASP.NET Core MVC 2.0, Adam Freeman describes adding the following to the Sports Store app:

  • Identity and an Identity database
    • Note: when running dotnet ef database update, you must specify a context since there is more than one. Example:
      • dotnet ef database update --context AppIdentityDbContext
  • Admin controller - Action methods for:
    • Add user
    • Delete user
    • Edit user
    • List users
  • Seed data class for Admin role and credentials
  • Account controller - Action methods for:
    • Login
    • Logout

Note that only the only users managed by Identity are the admins
